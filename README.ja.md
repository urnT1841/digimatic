# これは何?
これはミツトヨのデジタルキャリパーからの出力をPCで受けられるようにするツール(になる予定のもの)です。
ノギスからの信号は RaspberryPi Pico (実際は seeed RP2040) が受けて PCに送ります。

ノギス
  →
レベルシフタ(SN74LXC8T245PWR) をDip化
  →
RP Pico
   → 
PC (Linux / Windows)


## 現状
今の段階：
