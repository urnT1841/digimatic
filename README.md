# これは何?
これはミツトヨのデジタルキャリパーからの出力をPCで受けられるようにするツールです。

## 現状
今の段階では Sim状態です。
これはツール内で以下をやっているだけで,実機をつないだときの準備という段階です。
  - 乱数で 測定値もどきを生成
  - デジマチックのフレームを生成して送信
  - 受け取ってデコード → 表示

今後,実機との接続向けて開発していきます。
